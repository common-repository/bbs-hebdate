BB's plugin for displaying the hebrew date on your site

== Description ==

When activatied, substitutes the apropiate hebrew date (in hebrew) for the year, month and day parts of any date on your site.

== Installation ==

1. Upload `bb-hebdate.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
